¶ Formz

Formz aims to build web forms based on Xml Schemas. We know that that data-driven web apps are not very flexible, so the main scenario is to facilitate the creation of GUI's to xml-based configuration files.

It is also a simple opportunity to learn about many interesting things: - Angular JS - HTML enhanced for web apps. - karma - Spectacular test runner for Javascript. - Node.js - Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. - bower - A package manager for the web, html, css and javascript - docker - A simple documentation generator based on docco. - grunt - The Javascript task runner. - npm - Node.js package manager. - Travis CI - Continuous integration in the cloud. - Vagrant - Development environments made easy. - Ansible - Ansible is the easiest way to deploy, manage, and orchestrate computer systems you've ever seen, or so they say. ;) - Gemnasium - Gemnasium is an online tool to monitor your project dependencies.

¶ Getting Started

Theses instructions only work for Unix-based machines. Follow them if you want to hack away.

Clone Formz repository:

git clone git://github.com/jhosm/Formz.git

Install Vagrant.

Install ansible, at least v1.2.

Create your vagrant machine:

cd Formz
vagrant up

Fire up karma for continuous unit test execution:

vagrant ssh
cd /vagrant
grunt unitTests

Happy coding!
